使用说明

快速上手 JapAI，5 分钟搞定

1

获取 DeepSeek API Key

前往 DeepSeek 开放平台 注册账号并创建 API Key。

  • 在设置页面中输入你获得的 API Key（以 sk- 开头）
  • Key 仅保存在你电脑本地，不会上传到任何服务器
  • DeepSeek API 采用按量计费，每次出题 + 批改约花费 ¥0.01 左右，新用户通常有免费额度
2

输入本日语法笔记

将今天学到的语法内容粘贴到笔记框中。没有严格的格式要求，但建议按如下结构整理，AI 理解会更准确：

建议格式：语法点 / 接续 / 中文意思 / 例句
例如：〜てから / Vて形 + から / 做完…之后 / 朝ごはんを食べてから、学校に行きます。

也可以在教学材料中批量复制粘贴，AI 会自动提取语法点。选填「今日单词」可让 AI 在出题时融入新词汇。

3

开始练习，逐题翻译

点击「开始训练」后，AI 会根据你的笔记和所选级别生成翻译题：

  • 每道题会展示一个使用场景和一句中文，你需要将其翻译成日语
  • 在输入框中输入日语翻译，按 Enter 键提交
  • 如果有词汇提示，可以点击展开查看
  • 题目难度会根据你的表现动态调整
4

查看批改反馈

提交答案后，AI 会逐项分析你的翻译：

  • ✅ 正确的地方：你做得好的部分
  • ❌ 需要注意的地方：错误点、正确写法和详细解释
  • 💡 更自然的说法：让你的日语更地道的建议

得分 ≥ 8 分可以选择「加大难度」挑战更难题目；得分较低可以选择「换道同类题再练」巩固薄弱语法，或「修改重答」同一道题。

5

获取当日学习总结报告

完成全部题目后，系统会自动生成：

  • 统计仪表盘：答题数、平均分、高分题、薄弱项一目了然
  • 复习笔记（Markdown）：包含所有语法点、正确/错误分析和改进建议
  • 点击「下载 .md 文件」可将笔记保存到本地，方便日后复习
  • 可选择将本次语法点保存到知识库，系统会根据艾宾浩斯记忆曲线自动安排复习
