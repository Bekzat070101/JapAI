# JapAI — 项目需求文档

> 版本：v1.0 | 更新日期：2026-07-20

---

## 一、项目概述

JapAI 是一款运行在 Windows 上的本地日语语法闯关练习工具。用户将幕布笔记粘贴到工具中，AI 自动生成翻译题（中文→日语），用户逐题作答并获得即时批改反馈。所有题目完成后，AI 根据答题情况生成专属 Markdown 复习笔记。

---

## 二、目标用户

- 日语学习者（N5~N1 各级别）
- 日常使用幕布整理语法笔记
- 不要求编程能力，双击 bat 文件即可启动

---

## 三、核心功能

### 3.1 API Key 管理
- 首次启动时要求输入 DeepSeek API Key
- 存入 `config.json`，后续启动跳过此步骤
- 支持通过设置入口修改 API Key

### 3.2 能力级别选择
- 在粘贴笔记前选择 JLPT 级别：N5 / N4 / N3 / N2 / N1
- 默认选中 N4
- 级别影响出题难度、批改标准、超纲判断

### 3.3 笔记输入与出题
- 大文本框粘贴幕布语法笔记
- 点击"开始训练"→ AI 分析笔记中语法点并生成翻译题
- 出题数量：每个语法点 1~2 题，最少 5 题，最多 15 题
- 出题时自动参考 **已学习内容库**（`learned_content.json`），已学内容可融入题目作为复习

### 3.4 闯关答题
- 逐题展示，顶部显示进度（第 X / N 题）
- 题目包含：场景描述 + 中文句子 + 词汇提示
- 单行输入框输入日语翻译
- 支持回车键和按钮提交

### 3.5 即时反馈
- 正确部分（✅ 绿色标注）
- 错误部分（❌ 红色标注 + 语法解释）
- 修改建议（💡 更自然的日语说法）
- 得分（X.X / 10）
- 超纲内容标注（📌 拓展，答错不扣分，答对额外加分）

### 3.6 动态后续选项
- **得分 ≥ 8 分**："加大难度 🔥" / "下一题 →"
- **得分 < 8 分**："再练一次 🔄" / "跳过 →"
- "加大难度"分为 Lv1→Lv2→Lv3 三级递进

### 3.7 生成复习笔记
- 所有题目完成后自动触发
- 生成包含以下内容的 Markdown 文件：
  - 掌握良好的知识点
  - 薄弱环节与易错点
  - 错题集锦（原题 + 错误答案 + 正确答案）
  - 超纲内容拓展记录
  - 针对当前级别的后续复习建议
- 页面预览 + "下载 .md 文件"按钮

### 3.8 已学习内容追踪（新增）
- 每次完成答题后，自动提取所有涉及的语法点
- 更新 `learned_content.json`（语法点 + 学习时间 + 掌握程度 + 级别）
- 后续出题时，将已学内容告知 AI，使其：
  - 融入新题目作为复习（不算超纲）
  - 在综合题中混合已学语法点
  - 避免将已学内容标注为超纲

### 3.9 中途退出恢复
- 每答完一题保存 `progress.json`
- 下次启动检测到未完成进度 → 弹窗询问是否继续
- 支持"继续"或"重新开始"

---

## 四、非功能需求

### 4.1 运行环境
- Windows 10/11
- Python 3.9+
- 本地运行，无需云服务器
- 仅调用 DeepSeek API 时联网

### 4.2 数据隐私
- 所有数据存储在本地文件
- API Key、笔记、答题记录均不外传
- 仅题目内容和用户答案发送至 DeepSeek API

### 4.3 性能
- 页面加载 < 1 秒
- API 调用有加载动画（通常 3~10 秒）
- 不支持并发用户（单机单用户）

---

## 五、用户流程

```
启动程序
  → [首次] 输入 API Key → 保存 config.json
  → 检测 progress.json → [有] 询问是否继续
  → 选择 JLPT 级别
  → 粘贴笔记 → 点击"开始训练"
  → 逐题作答 → 提交 → 查看反馈 → 选择下一步
  → 全部完成 → 生成复习笔记 → 预览 + 下载
  → 更新 learned_content.json
```
