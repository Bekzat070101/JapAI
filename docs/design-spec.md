# JapAI — UI 设计规范

> 版本：v1.0 | 更新日期：2026-07-20

---

## 一、整体风格

**Apple 果味** — 干净、通透、留白充足、毛玻璃质感。

- 像 macOS 原生应用，但运行在 Windows 上
- 非聊天软件，是专注的练习工具 / 考试界面
- 卡片、按钮、输入框都有细腻的轻拟物质感

---

## 二、颜色系统

### 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#B3D9FF` | 柔和蓝，按钮、选中态、进度条 |
| 主色深 | `#8EC5FF` | hover / active 态 |
| 辅色背景 | `#E8F4FD` | 浅蓝背景 |
| 页面背景 | `#F5F7FA` | 整体页面底色 |
| 卡片背景 | `rgba(255,255,255,0.7)` | 毛玻璃卡片 |
| 卡片纯白 | `#FFFFFF` | 无毛玻璃时使用 |

### 语义色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正确 / 通过 | `#34C759` | 绿色，正确部分标记 |
| 错误 / 失败 | `#FF3B30` | 红色，错误部分标记 |
| 建议 / 信息 | `#007AFF` | 蓝色，修改建议 |
| 超纲 / 拓展 | `#FF9500` | 橙色，📌 拓展标签 |
| 警告 / 注意 | `#FFCC00` | 黄色 |

### 文字颜色

| 用途 | 色值 |
|------|------|
| 主文字 | `#1D1D1F` |
| 次文字 | `#6E6E73` |
| 辅助文字 | `#AEAEB2` |
| 白色文字 | `#FFFFFF` |

## 七、毛玻璃


## 八、动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 屏幕切换 | fadeIn + slideUp | 0.3s ease-out |
| 按钮 hover | translateY(-1px) + 阴影加深 | 0.2s ease |
| 按钮 active | translateY(0) + 阴影减弱 | 0.1s ease |
| 卡片出现 | fadeIn | 0.4s ease-out |
| 反馈区域展开 | max-height 过渡 | 0.3s ease |
| 加载动画 | 旋转 spinner | 持续 |

## 十、组件规格

### 级别选择按钮
- 5 个并排胶囊形按钮
- 未选中：白色背景 + 浅灰边框
- 选中：`#B3D9FF` 背景 + 无边框 + 轻微放大
- 尺寸：100px × 40px

### 主按钮（主要操作）
- `#B3D9FF` 背景 → hover `#8EC5FF`
- 白色文字，600 字重
- padding: 12px 28px

### 次按钮（次要操作）
- 透明背景 + `#B3D9FF` 边框
- `#1D1D1F` 文字
- padding: 10px 24px

### 文本输入框
- 白色背景 + 浅灰边框 `#D1D1D6`
- focus 时边框变为 `#B3D9FF`
- 大文本框最小高度 200px
- 单行输入框高度 44px

### 进度条
- 高度 6px，圆角 3px
- 背景 `#E5E5EA`
- 填充 `#B3D9FF`，过渡 0.3s

### 得分显示
- 大号数字 + 圆环或简单分数显示
- ≥ 8 分绿色，5~7 分橙色，< 5 分红色
